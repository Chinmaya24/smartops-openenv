\# SmartOps OpenEnv 🚀



A multi-agent reinforcement learning environment for real-world customer support automation.



\## 🔥 Features

\- Multi-agent system (triage, response, escalation, manager)

\- RL training using PPO

\- LLM-powered reasoning (Hugging Face)

\- Real-world task simulation (email support)



\## 🧠 Tech Stack

\- Python

\- Stable-Baselines3 (PPO)

\- Hugging Face Transformers

\- Gymnasium
